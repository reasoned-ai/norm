Neural Object Relational Models (NORM) is a probabilistic logical programming language based on neural networks.

Overview
=======================================

Features
---------

- **Compositional**: composing machine learning models through a logical program where each logical function is a
  neural network module.
- **Conversational**: reasoning with Norms to improve them in a conversational environment.
- **Communal**: sharing Norms to help everyone in the community.
